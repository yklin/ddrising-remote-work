
## VENV installation

```
pip install -c requirements.txt
```

## HELP

```
python web-crawler.py -h

usage: web-crawler.py [-h] [-s START] [-e END] [-o OUTPUT]

AAPL crawler

optional arguments:
  -h, --help            show this help message and exit
  -s START, --start START
                        start date: default: 10 days ago
  -e END, --end END     end date, default: today
  -o OUTPUT, --output OUTPUT
                        output file
```

## RUN

```
python web-crawler.py
```
